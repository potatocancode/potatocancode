## 主要專案

| **[potatocancode-web](https://github.com/potatocancode/potatocancode-web)** | 工作室網站與後台管理，含上面 20 個範例 | Next.js 16 · React 19 · Supabase |
| **[studio07-demos](https://github.com/potatocancode/studio07-demos)** | 上面三組範例的原始碼 | TypeScript · Expo · 單檔 HTML |

**前端** TypeScript · React · Next.js (App Router) · Tailwind CSS · Framer Motion
**行動** React Native · Expo · Expo Router
**後端** Supabase (Postgres · Auth · Storage · RLS) · Drizzle ORM · Neon
**其他** C++ 系統程式 · Playwright · Vitest · Vercel
